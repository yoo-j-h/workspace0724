문단을 나누는 태그 : p, pre

문단을 나누는 태그로는 p태그와 pre태그가 있다.
p태그는 줄바꿈을 하고자한다면 별도의 태그를 작성해야한다.
그리고 공백은 한개의 공백만 표시하기 때문에
별도의      문구(엔티티코드)를 기술해야한다.
< > & 등등.... 

pre태그는 시작태그에서부터 종료태그까지 입력된 내용을 작성한 그대로 표현하는 태그
줄바꿈이라던가         여러개의 공백을 그대로 인식함.
b: 글자를 굵게표시하는 태그

string : 글자를 굵게표시하는 태그

em : 글자를 기울여 보여주는 태그

i : 글자를 기울여 보여주는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

기본글자에 sub : 아래첨자를 나타내는 태그 sup : 윗 첨자를 나타내는 태그

태그를 중첩해서 사용하는 건 당연하게 가능
굵고 기울인 글자
취소선, 형광팬
